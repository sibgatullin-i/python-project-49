## Project description

 _A project of 5 logical mini-games, which are built on the principle of "question - answer"._

 _**brain-even** - is number odd or even?_

 _**brain-calc** - human calculator_

 _**brain-gcd** - greatest common divisor_ [wiki](https://en.wikipedia.org/wiki/Greatest_common_divisor)

 _**brain-prime** - is it prime?_

 _**brain-progression** - fill in blanks in progression_

## Minimum requirement 

 _Python 3.10_

## Installing

 _To install use: ```make install```_

 _To build use: ```make build```_

 _To package install use: ```make package-install```_

## Game launch

 _To start the game, type: ```make 'game name'``` (game names are in the description)_
